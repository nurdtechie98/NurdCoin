# NurdCoin
My attempt to implement decentralized block-chain by creating a over-simplified Bitcoin like crypto-currency in python using flask named **NurdCoin**

## Features Implemented:
* Decentalized BlockChain
* Proof Of Work
* Consensus Algorithm
* Mining Commission
